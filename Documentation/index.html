A) Requirements - top

if you want to run this web application in your machine, you must have these tools setup on your machine.

  1. Visual Studio 2017 comes with .NET CORE 2.1 SDK
  2. MS SQL Server or Local Server (which comes with Visual studio)

B) Project Structure - top

If you extract the Invoice.rar file, you will find a folder call "Project file" there, open the folder and you'll see a visual studio solution called "Invoice". Open that file in Visual Studio and it'll see like this below

There are two class library and aspnet core mvc project in this solution

  1. EasyHRM.Core
  2. EasyHRM.Data
  3. EasyHRM.Web

C) Change Database Connection - top

In EasyHRM.Web, there is an appSettings.json, open that file. and find the "connectionStrings" section there, you'll see like below 

	
"ConnectionStrings": {
     "Default": "Server = .\\SQLexpress;Database= invoiceCoreDb;Integrated Security=True; "
},

just change the connection string as your SQL server instance name on Data Source, like below

	
  "Default": "Server = YOUR_SQL_SERVER_INSTANCE;Database= invoiceCoreDb;Integrated Security=True; "

D) Migrate and Run the Application - top

Go to tools > Nuget package manager > Select 'EasyHRM.Data' > package manager console then write Update-Database

Now, if your connection is okay, it'll generate the database e for you! Now you'll able to login as a admin
